# d2python | 动手学Python

强调动手实践的python教程

## 使用项目

> 需要[安装xlings环境](https://github.com/d2learn/xlings)

### 下载

> 下载d2python到当前目录

```bash
xlings drepo d2python
```

注: `xlings drepo`可以查看项目信息

### 打开项目书籍

```bash
xlings book
```

### 运行项目练习

> 运行d2python项目中的pylings代码练习

```bash
cd d2python
xlings pylings
```
